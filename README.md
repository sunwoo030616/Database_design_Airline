# ✈️ Airline Reservation System (항공권 예약 시스템) A+받음!!!!!!!!!!!

## 🌟 프로젝트 개요 (Overview)

이 프로젝트는 사용자 관리, 항공편 스케줄링, 실시간 예약 및 결제 처리를 포함하는 종합 항공권 예약 시스템입니다. 특히 수요에 따라 요금을 조정하는 **동적 가격 책정(Dynamic Pricing)** 로직을 핵심적으로 구현하고, 정교한 **관계형 스키마 설계**를 기반으로 데이터의 무결성을 확보합니다.

---

## ✨ 주요 기능 (Key Features)

| 카테고리 | 상세 기능 |
| :--- | :--- |
| **회원 관리** | 일반 고객(`Customer`)과 관리자(`Admin`) 역할을 분리하여 관리하며, 고객은 마일리지와 등급 정보를 가집니다. |
| **운항/좌석** | 공항, 노선, 항공기 기체, 좌석 등급 및 상태 정보를 체계적으로 관리합니다. |
| **실시간 예약** | 동일 항공편의 중복 좌석 예약을 방지하는 강력한 데이터 무결성 로직을 적용했습니다. |
| **가격 정책** | **동적 운임 로그 (`DynamicFareLog`)**를 통해 예약, 취소 등의 이벤트에 따른 운임 변동을 투명하게 추적하고 관리합니다. |
| **분석** | 노선, 항공기, 월별 좌석 등급에 따른 총수익을 집계하는 분석용 테이블(`RevenueAnalysis`)을 제공합니다. |
